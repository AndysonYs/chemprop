Chemprop — Machine Learning for Molecular Property Prediction

Introduction

This website can be used to predict molecular properties using a Message Passing Neural Network (MPNN). In order to make predictions, an MPNN first needs to be trained on a dataset containing molecules along with known property values for each molecule. Once the MPNN is trained, it can be used to predict those same properties on any new molecules.

Training

To train an MPNN, first upload a dataset of molecules with known properties on the Data page. Then go to the Train page and start training your model. Once training is complete, a model checkpoint will be saved. You can view, download, and delete model checkpoints on the Checkpoints page.

Predicting

To make property predictions using, go to the Predict page, select the trained model checkpoint you want to use, and then upload or paste the molecules you would like to make predictions on.


Working Remotely

If you wish to train or predict on a remote server — for instance, if a remote server has GPUs you would like to use — you can use SSH port-forwarding to run training/predicting on the remote server while viewing the website it locally. To do so, follow these instructions:

  1. Connect to the remote server: ssh <remote_user>@<remote_host>
  2. On the remote server, start the website: python web.py
  3. On your local machine, set up port-forwarding: ssh -N -L 5000:localhost:5000 <remote_user>@<remote_host>
  4. On your local machine, open a web browser and navigate to localhost:5000
